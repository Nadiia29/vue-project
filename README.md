##  📋 Опис
Цей проєкт - це односторінковий додаток на Vue 3 + TypeScript, який дозволяє користувачам:

* Ввести особисті дані (ім’я, email, пароль)

* Обрати улюблену веб-технологію (HTML, CSS, JavaScript, Vue, React)

* Обрати улюблений AI-інструмент (ChatGPT, Copilot, Gemini тощо)

* Надіслати форму та переглянути збережені дані

* Керувати списком користувачів через дашборд (редагування, видалення)
